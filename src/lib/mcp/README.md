# MCP Integration Library

This directory contains utilities and helpers for integrating with Model Context Protocol (MCP) servers.

## Structure

- `supabase-helpers.ts` - Helper functions for Supabase MCP operations
- `playwright-helpers.ts` - Helper functions for Playwright MCP operations
- `types.ts` - TypeScript types for MCP interactions

## Usage

These helpers can be used in your application code to interact with MCP servers programmatically, or to build features that leverage MCP capabilities.

## Future Extensions

As Asura grows, this directory will contain:
- Custom MCP server implementations
- Shared utilities for AI-assisted operations
- Type definitions for MCP resources and tools
